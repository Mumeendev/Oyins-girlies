# Oyin's Girlies - Boutique

A modern shopping website for cosmetics, clothes, skincare, and hair care.

## Setup Instructions

1.  **Hero Image:** Go to your **Pictures** folder, find the image you want to use for the hero section, and copy it into this project folder.
2.  **Rename Image:** Rename that image to `hero-image.jpg` so the website can find it.
3.  **Hosting on GitHub Pages:**
    *   Initialize a git repository: `git init`
    *   Add your files: `git add .`
    *   Commit: `git commit -m "Initial commit"`
    *   Push to a new GitHub repository.
    *   Go to **Settings > Pages** on GitHub and set the branch to `main`.

## Design Theme
- **Colors:** Pink (#ff007f), Purple (#800080), and Black (#1a1a1a).
- **Structure:** 
  - `index.html`: Hero landing page.
  - `shop.html`: Category selection page.
  - `style.css`: Unified styling.
  - `script.js`: Interactive elements.
